# BCF Chatbox MVP – Client Demo Script

A short, step-by-step walkthrough for presenting the Dutch chatbox MVP to the
client. Run through the steps in order and tick each as you go.

## Demo flow

- [ ] 1. Open the Dutch website page
- [ ] 2. Open the chatbox (floating button, bottom-right)
- [ ] 3. Show the welcome message and the quick-reply buttons
- [ ] 4. Click **“Wat is de minimale bestelhoeveelheid?”** and verify the approved answer (orders from 20 pieces)
- [ ] 5. Click **“Wat is de leveringstermijn?”** and verify the approved answer (delivery in 2–4 weeks)
- [ ] 6. Type **“Wat kost 500 stuks?”** and verify the chatbox does **not** invent a price
- [ ] 7. Show that it offers to forward the question to BCF Confection
- [ ] 8. Choose **“Ja, stuur mijn vraag door”**
- [ ] 9. Fill in the handoff form with **test data** (name, test email, question)
- [ ] 10. Show the confirmation screen that appears **before** sending
- [ ] 11. Explain that production email routing will only be enabled **after client approval**
- [ ] 12. Show privacy behavior: conversation is session-only, no full history is sent
- [ ] 13. Show the unsupported-question fallback with an unrelated question (e.g. “Wat is de hoofdstad van Frankrijk?”)
- [ ] 14. Confirm the plan: Dutch MVP first, French version later
